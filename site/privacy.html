Политика обработки персональных данных

Настоящая Политика определяет порядок обработки и защиты персональных данных Пользователей сервиса VPN_GPT.

1. Общие положения

1.1. Персональные данные — информация, предоставленная Пользователем добровольно при взаимодействии с Telegram‑ботом сервиса VPN_GPT.

1.2. К таким данным могут относиться: имя пользователя Telegram, идентификатор (ID), сообщения и иные сведения, необходимые для корректной работы бота.

2. Цели обработки

2.1. Данные обрабатываются исключительно для оказания консультаций и предоставления рекомендаций по выбору VPN‑сервисов.

2.2. Данные не передаются третьим лицам и не используются для рекламы.

3. Защита персональных данных

3.1. Создатель проекта принимает все необходимые меры для защиты данных от несанкционированного доступа и утечки.
